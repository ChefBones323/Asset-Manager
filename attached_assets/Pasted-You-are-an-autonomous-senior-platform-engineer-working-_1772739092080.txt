GLOBAL ARCHITECTURE RULES (GUARDRAIL)

This subsystem uses deterministic event sourcing.

You must preserve these invariants:

1. approval required before execution
2. lease ownership required for state mutation
3. only one active lease per job
4. deterministic manifests
5. domain isolation
6. full audit logs
7. replayable state from event logs

Never write state directly to projection tables.

All mutations must originate from:

ExecutionEngine → EventStore → ProjectionEngine

Never replace the event architecture with CRUD patterns.
