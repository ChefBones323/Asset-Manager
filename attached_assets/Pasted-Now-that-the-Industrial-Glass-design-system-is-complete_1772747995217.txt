GlassCard

CMD + K
CTRL + K

components/palette/CommandPalette.tsx

Floating glass modal
Keyboard navigation
Real-time fuzzy search
Command preview
Keyboard execution

Command categories:

Governance
Feed
Trust
Infrastructure
Events
Diagnostics

Example commands:

simulate feed policy

show active workers
inspect events
explain feed ranking
replay projections
system health

EVENT PULSE PANEL

components/events/EventPulsePanel.tsx

blue = content
green = trust
purple = governance
orange = workers
red = errors
