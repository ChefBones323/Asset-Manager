You are building a production-grade operator interface called CIVIC MISSION CONTROL for a deterministic event-sourced Social Civic Infrastructure Platform.

This platform is NOT a typical social media app.
It is a governed civic infrastructure system with:

• Event Store (append-only ledger)
• Execution Engine
• Worker System
• Governance Proposal Engine
• Feed Ranking Policies
• Trust Graph Reputation System
• Deterministic projections

Your task is to build the complete frontend control environment that allows operators, moderators, developers, and civic administrators to control and observe the system.

The UI must feel like a hybrid between:

NASA mission control
Bloomberg terminal
VSCode command palette
Raycast productivity tools

The interface must be keyboard-first, event-driven, and extremely transparent.

The result should feel like the operating system for civic information infrastructure.

⸻

PRIMARY OBJECTIVES

Build a full Mission Control Interface including:

• Civic Command Palette (keyboard command interface)
• Mission Control dashboard
• Governance console
• Feed ranking debugger
• Trust graph visualizer
• Event explorer
• Infrastructure worker monitor

The system must allow operators to:

propose governance actions
simulate feed policies
inspect event streams
debug feed ranking
monitor workers
analyze trust networks

Everything must be observable and explainable.
